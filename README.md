## **Opis projektu**
*Cel projektu:* Zapakowanie samochodu w taki sposób, aby zmieścić jak największą ilość paczek.  
*Wymagania:* 
- przestrzeń zajmowana przez paczki nie może być większa od maksymalnej przestrzeni ładunkowej pojazu (5m3),
- przestrzeń zajmowana przez paczki musi być jak najbliższa połowy maksymalnej przestrzeni ładunkowej pojazdu,
- maksymalny udźwig przewożonych paczek wynosi 1500kg,
- ilość paczek wynosi 100000.
